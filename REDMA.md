Autor: Haz Vera Justyn Mateo\
Fecha: 29/1/2026\
Materia: Programación Orientada a Objetos\

**Descripción**\
Dashboard interactivo para organizar y gestionar proyectos y ejercicios de Programación Orientada a Objetos. Permite navegar entre diferentes unidades, visualizar código fuente y ejecutar scripts de Python de manera organizada.

**Características**
1. Navegación por carpetas y archivos Python
2. Visualización de código directamente desde el dashboard
3. Ejecución de scripts en terminal separada
4. Navegación intuitiva con menús jerárquicos
5. Organización por unidades temáticas

**Modificaciones Realizadas**

Basado en el proyecto original de snogales-uea, se realizaron las siguientes mejoras:

* Personalización del menú principal con títulos descriptivos y emojis
* Soporte para archivos directos - ahora muestra archivos .py en carpetas sin subcarpetas
* Sistema de colores con colorama para mejor experiencia visual
* Mejoras en mensajes de error más claros y descriptivos
* Adaptación a estructura de carpetas personalizadas:
\
EjemplosMundoReal_POO - Ejemplos prácticos de POO
POO-python - Fundamentos y ejercicios de Python

Requisitos\
Python 3.x\
xterm (Linux) o cmd (Windows) para ejecución de scripts
