# GUI - Gestor de tareas (Tkinter)

Aplicacion simple de escritorio para gestionar tareas:
- Anadir una tarea desde el campo de texto
- Marcar una tarea como completada
- Eliminar una tarea seleccionada
- Interaccion por eventos (botones, Enter, doble clic y tecla Delete)
- Persistencia automatica en JSON para conservar tareas entre ejecuciones
## Archivos
- `todo_app.py`: aplicacion principal.
- `tests/test_todo_app.py`: pruebas unitarias de la logica (`TodoStore`)
- `todo_tasks.json`: archivo creado automaticamente con las tareas guardadas
